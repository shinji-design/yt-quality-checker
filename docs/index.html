あなたの動画、収益化に通る品質ですか?

YouTube動画の最初の3分を分析して、合格水準かをお伝えします

使い方は3ステップ

1. 下のリンクをブラウザのブックマークバーにドラッグして登録
2. 分析したいYouTube動画ページを開く（推奨：3ヶ月以上前の動画）
3. 登録したブックマークをクリック → 約4分待つ → 「📊 詳細レポートを見る」ボタンを押す

📌 動画クオリティチェック（ドラッグして保存）

↑ このボタンをブックマークバーへドラッグしてください（クリックでは動きません）

📋 結果でわかること

  • 合格水準の総合スコア（100点満点）
  • 5項目スコア（動きのバランス／止まっている時間／場面切替／動きの種類／集中度）
  • 5段階判定（合格水準・ほぼ合格・もうすぐ合格・要改善・大幅作り直しが必要）
  • 優先度付きの具体的な改善アクション（最大3件）
