❓ 拡張機能とは? 安全ですか?

Chrome拡張機能は、Chromeブラウザに小さな機能を追加する公式の仕組みです。
このツールの拡張機能は あなたのChromeでYouTube動画を再生してフレームを測る ためだけに使われます。
・動画データを外部サーバに送ることはありません
・あなたのGoogleアカウント情報にもアクセスしません
・いつでも Chrome の拡張機能管理画面から削除できます

📖 詳しいインストール手順
  1. 「📥 拡張機能をインストール」ボタンを押す
  2. Chrome ウェブストアのページが開く
  3. 「Chromeに追加」ボタンを押す
  4. 確認ダイアログで「拡張機能を追加」を押す
  5. このページに戻って「🔄 ページを再読み込み」を押す
🛠 開発者モードで読み込む（公開前テスト用）
  1. Chromeのアドレスバーに chrome://extensions/ を入力
  2. 右上の「デベロッパーモード」をオンにする
  3. 「パッケージ化されていない拡張機能を読み込む」をクリック
  4. このリポジトリの extension/ フォルダを選択
  5. このページに戻って再読み込み
