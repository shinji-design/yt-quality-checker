🔐 プライバシーポリシー

最終更新日：2026年4月28日

結論：このツールは、あなたの動画データを外部に送信しません。
分析はすべてあなたのChrome内で完結します。

1. 適用範囲

本プライバシーポリシーは、以下のサービスに適用されます。

  • Webサイト：https://shinji-design.github.io/yt-quality-checker/
  • Chrome拡張機能：「動画クオリティ・チェッカー」

2. 取得する情報

本サービスは、ユーザー個人を識別する情報を一切取得しません。

2-1. 拡張機能が一時的に処理する情報

動画分析機能を使用する際、拡張機能は以下の情報をあなたのChrome内でのみ処理します。

  • あなたが入力したYouTubeチャンネルURLまたは動画URL
  • YouTube動画の最初の180秒のフレーム画像（80×45ピクセルに縮小）
  • 動画タイトル（結果画面の表示用）

これらの情報は、外部サーバには一切送信されません。分析結果の表示が完了した時点で、ブラウザのメモリから消去されます。

2-2. Webサイトのアクセス情報

本Webサイトは GitHub Pages 上でホスティングされており、GitHub のサーバが標準的なアクセスログ（IPアドレス、アクセス日時、参照元）を記録する場合があります。これらは GitHub のプライバシーポリシーに従って管理されます。

  • GitHub プライバシーステートメント：https://docs.github.com/ja/site-policy/privacy-policies/github-general-privacy-statement

3. 取得しない情報

本サービスは、以下の情報を一切取得しません。

  • あなたのGoogleアカウント情報（メールアドレス、ユーザー名など）
  • YouTubeのログイン情報や視聴履歴
  • Cookieやローカルストレージに保存された個人情報
  • 他のWebサイトの閲覧履歴
  • ブラウザのブックマークやパスワード
  • クレジットカード情報や決済情報

4. 情報の用途

取得した情報（前述2-1）は、以下の目的のみに使用されます。

  • あなたが指定したYouTube動画の動きの特徴を分析すること
  • 分析結果（5項目スコア・総合判定・改善アドバイス）を画面に表示すること

5. 情報の第三者提供・販売

取得した情報を第三者に提供・販売することは一切ありません。

6. データの保存

本サービスは、サーバ側にユーザーデータを保存するデータベースを持ちません。動画分析中の一時データは、結果表示後に自動的に消去されます。

7. 拡張機能の権限について

本拡張機能は、Chrome の以下の権限を要求します。

  • tabs：分析用にYouTube動画ページを新しいタブで開き、分析完了後にそのタブを閉じるため
  • scripting：YouTube動画ページに分析スクリプトを注入し、video要素から1秒ごとにフレームを取得するため
  • https://*.youtube.com/*：YouTube動画のvideo要素にアクセスするため
  • https://shinji-design.github.io/*：本ツールのWebサイトと拡張機能の間で分析依頼と結果のやり取りを行うため

これらの権限は、上記の目的以外には使用しません。

8. お子様の利用について

本サービスは、収益化審査を控えた動画クリエイター向けの業務支援ツールです。13歳未満のお子様による利用は想定していません。

9. ポリシーの変更

本プライバシーポリシーは、必要に応じて改定されることがあります。重要な変更がある場合は、本ページに掲載してお知らせします。

10. 免責事項

本サービスの分析結果は、YouTube収益化審査の結果を保証するものではありません。「動画の動き方」の特徴のみを判定するものであり、内容の善し悪し・著作権・規約への適合などはチェックしていません。あくまで参考情報としてご利用ください。
